# 云端订单管理系统

基于Supabase云存储的现代化订单管理系统，支持多设备实时同步。

## 🌟 功能特性

### ☁️ 云端存储
- **Supabase集成**：使用PostgreSQL数据库存储订单数据
- **实时同步**：多设备间自动同步订单状态
- **离线支持**：网络中断时自动使用本地存储
- **数据备份**：支持导出JSON格式备份文件

### 🔒 安全特性
- **密码保护**：派单和查看订单需要密码验证
- **数据加密**：敏感数据在传输过程中加密
- **访问控制**：基于Supabase的行级安全策略

### 📊 订单管理
- **创建订单**：单名、派单人、价格、备注、优先级、状态
- **状态跟踪**：待处理 → 处理中 → 已完成
- **智能筛选**：多条件搜索和过滤
- **批量操作**：一键导出、同步、清空

### 🎨 用户体验
- **现代化UI**：玻璃态设计，渐变色彩
- **流畅动画**：弹窗滑动效果，平滑过渡
- **响应式设计**：完美适配桌面、平板、手机
- **实时统计**：仪表板显示关键数据指标
